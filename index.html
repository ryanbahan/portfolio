Featured Work

What's New?

What's new is a simple & efficient news application built in React. It fetches recent news stories via the News API, and serves up news in entertainment, health, technology, and science. Users can also search by topic, or view location-based news, made possible via the reverse geocoding functionality of Mapquest API.

FitLit

A fitness tracking dashboard displaying metrics from a large data set. Metrics include daily values of steps, minutes active, flights climbed, miles traversed, hours of sleep, quality of sleep, and hydration. Main focuses include OOP design, writing modular/reusable code that follows the Single Responsibility Principle, and implementing a robust test suite using TDD.

Trip Advicer

TripAdvicer is a Trip Advisor clone, built around creating a delightful user experience for both travelers and agents. It features different views for both the traveler and agent (including a revenue dashboard for agents), and updates/persists data in a remote server via HTTP requests to multiple endpoints.
